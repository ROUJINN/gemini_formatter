# Gemini 文本处理器（Web版）

> 以下为AI生成

简体中文说明文档，配合项目中的前端页面使用：本地打开 `index.html` 即可进行文本清理与快速复制。

**功能简介**
- 将粘贴的原始文本按规则删除指定内容并输出处理结果。
- 支持两类规则：普通文本（exact match）与正则表达式（Regex，内部自动使用 `gm` 标志）。
- 提供默认规则（例如：去除 `**`、空行、某些模式），可通过 UI 添加/删除自定义规则。
- 支持快捷复选（如移除 Markdown 标题 `^#+\s`），处理后结果会自动复制到剪贴板并在页面显示状态提示。

**快速使用**
1. 在浏览器中打开：
https://roujinn.github.io/gemini_formatter/
2. 在上方文本框粘贴或输入需要处理的文本。
3. 在「当前生效的删除规则」中查看/管理规则；可通过「普通文本」或「正则」添加新规则。
4. 使用快捷勾选快速开启常用规则（例如去掉 Markdown 标题）。
5. 点击「删除指定字符并复制到剪贴板」按钮，处理结果会显示在下方并复制到剪贴板。

**注意事项与实现细节**
- 正则规则会在代码中使用 `new RegExp(pattern, 'gm')` 创建，因此 `^` 可以匹配每一行行首，替换为全局多行模式。
- 普通文本规则使用 `split(pattern).join('')` 进行全局删除，适用于精确串匹配。
- 添加正则时请确保正则语法正确，前端会捕获并在控制台打印错误，但不会阻止其它规则生效。

**示例**
- 去除 Markdown 标题：勾选“Markdown 标题 (#+ 空格)”的快捷选项，然后粘贴含 `# 标题` 的文本并处理。
